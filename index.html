Arcade Poker

Jonathan Yuen

Fun   new   way   to   play   poker

Simple   Mechanics

Drag  over  5  cards  to  make  a  hand,  earn  points  based  on  your  hand  strength.

Global   Leaderboard

Compete  against  players  and   put  your  name  on  the  leaderboard!

Compete   with   Friends

Play  against  your  friends  in  real  time  with  the  two  multiplayer  modes.

How it's built

Front End

Powered  by  the  latest  front-end  technology  from  Facebook,  built  entirely  using  React  Native

Realtime

Firebase  provides  a  powerful,  fast,  and  flexible  data  storage  for  highscores  and  enables  realtime  communication  between  devices.

Development

Build  versions   and  updates  are  managed  through  Xcode  and  Android  Studio.  Test  were  written  using  Jest.
